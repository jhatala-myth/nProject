Prerequisites:
--------------
â€¢ Docker Desktop (Windows/Mac) or Docker Engine (Linux)
â€¢ Docker Compose

Step 1: Install Docker
----------------------
Windows/Mac: Download from https://www.docker.com/products/docker-desktop
Linux: 
  $ curl -fsSL https://get.docker.com -o get-docker.sh
  $ sudo sh get-docker.sh

Step 2: Extract Files
----------------------
Extract all the project files to a folder, for example:
  C:\project-manager\  (Windows)
  ~/project-manager/   (Mac/Linux)

Step 3: Start the Application
------------------------------
Open terminal/command prompt in the project folder:

Quick Start (Linux/Mac):
  $ chmod +x start.sh
  $ ./start.sh

Manual Start (All platforms):
  $ docker-compose up -d

Step 4: Access the Application
-------------------------------
Open your browser and go to:
  http://localhost:5000

Step 5: Stop the Application
-----------------------------
  $ docker-compose down

Problem: Port 5000 already in use
Solution: 
  Option A - Change port in docker-compose.yml:
    ports:
      - "8000:5000"  # Use port 8000 instead
    Then access: http://localhost:8000
  
  Option B - Stop other services using port 5000

Problem: Docker container won't start
Solution:
  1. Check logs: docker-compose logs web
  2. Rebuild: docker-compose down && docker-compose up --build -d
  3. Check Docker is running: docker ps

Problem: Database locked error
Solution:
  Stop all instances: docker-compose down
  Then restart: docker-compose up -d

Problem: Permission denied on start.sh
Solution:
  Make it executable: chmod +x start.sh

Problem: Cannot install Python packages
Solution:
  1. Update pip: pip install --upgrade pip
  2. Use Python 3.11+: python3.11 -m pip install -r requirements.txt

Problem: Page won't load
Solution:
  1. Check if app is running: docker-compose ps
  2. Check logs: docker-compose logs -f
  3. Try accessing: http://127.0.0.1:5000

Docker Commands:
----------------
Start application:       docker-compose up -d
Stop application:        docker-compose down
View logs:              docker-compose logs -f
Restart application:     docker-compose restart
Rebuild container:       docker-compose up --build -d
Check status:           docker-compose ps
Enter container:        docker-compose exec web sh

Database Commands:
------------------
Access database (Docker):
  $ docker-compose exec web sh
  $ sqlite3 /app/data/projects.db

Access database (Local):
  $ sqlite3 projects.db

View tables:
  sqlite> .tables
  sqlite> SELECT * FROM projects;
  sqlite> .quit
